Definição: HTML

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto, em português) e é a linguagem utilizada para criar páginas da web. O HTML é composto por elementos que são marcados com tags, que indicam ao navegador como exibir o conteúdo na página, como por exemplo, cabeçalhos, parágrafos, imagens, links, entre outros.

As tags HTML são escritas usando colchetes angulares ("<" e ">"), com o nome da tag entre eles. Algumas tags podem ter atributos que especificam características adicionais, como a cor de um texto ou a largura de uma imagem.

O HTML é uma linguagem fundamental para o desenvolvimento de páginas da web e é frequentemente combinado com outras tecnologias, como CSS (Cascading Style Sheets) e JavaScript, para criar páginas web interativas e visualmente atraentes.

Tag head

A tag

<head></head> é usada para incluir informações sobre o documento HTML que não são exibidas na página da web. Essas informações podem incluir o título da página, a codificação de caracteres, os estilos CSS e os scripts JavaScript. A tag <head></head> deve estar localizada antes da tag <body> no documento HTML.

<h1> a <h6> O h1 é usado para o título principal da página, enquanto os níveis subsequentes são usados para subtítulos e títulos secundários.
